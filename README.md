# Validador de Imagens Faciais
Ferramenta de validação automática de imagens faciais, criada para verificar conformidade técnica e qualidade visual de fotos enviadas a sistemas de identificação.

## Status do projeto
Em desenvolvimento.

## Motivação
Durante o estágio, foi identificado um problema recorrente no recebimento de imagens que não atendiam aos critérios mínimos exigidos pelo sistema, como tamanho inadequado, baixa resolução, ausência de cor e outros desvios de padrão.

Este projeto surgiu como uma proposta para automatizar a triagem inicial dessas imagens, reduzindo erros manuais e melhorando a qualidade do material recebido.

## Objetivo
Automatizar a validação inicial de fotos enviadas a sistemas de identificação, com base em critérios objetivos e passíveis de análise automatizada.

A proposta é verificar se a imagem atende a requisitos básicos como:

- tamanho máximo do arquivo
- formato permitido
- resolução mínima
- presença de cor
- metadados EXIF para fins de teste, exploração e segurança

## Funcionalidades implementadas
Atualmente o projeto já realiza:

- validação de tamanho do arquivo
- validação de formato da imagem
- validação de resolução mínima
