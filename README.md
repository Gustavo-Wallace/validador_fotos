# Validador de Imagens Faciais
Ferramenta de validação automática de imagens faciais, destinada a verificar conformidade técnica e qualidade visual de fotos enviadas ao sistema, com base em critérios objetivos como tamanho do arquivo, resolução, formato, presença de cor e outros critérios de qualidade.

## Objetivo
Este projeto foi criado com o objetivo de automatizar a validação inicial de imagens utilizadas em sistemas de identificação, reduzindo erros manuais e melhorando a qualidade das fotos recebidas.

A proposta é rerificar automaticamente se a imagem atende a requisitos básicos como:

- Tamanho máximo do arquimo
- Formatos permitidos
- Resolução mínima
- Imagem colorida
- Metadados EXIF para fins de teste, exploração e segurança
